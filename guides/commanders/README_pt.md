# Guia de Comandantes Coletores

**[Español](README_es.md) | [English](README_en.md) | Português | [Tiếng Việt](README_vi.md) | [Bahasa Indonesia](README_id.md) | [Français](README_fr.md)**

Guia rápido para evoluir comandantes de coleta com eficiência.

## Ordem recomendada

1. Sarka
2. Caio Mario
3. Centuriao
4. Joana d'Arc
5. Matilde / Cleopatra / Ishida / Seondeok

## Progressão sugerida

1. Evoluir Sarka e Caio até o nivel 20.
2. Evoluir Centuriao até o nivel 20.
3. Na Prefeitura 17, evoluir Joana até o nivel 20.
4. Evoluir os 5 coletores até o nivel 27.

## Metas de capacidade

- Meta intermediária: 630000 por marcha.
- Meta final: 1260000 por marcha.

## Recomendações principais

- Use os coletores como comandantes principais.
- Priorize talentos de coleta antes dos de combate.
- Treine tropas para aumentar a capacidade de carga.
