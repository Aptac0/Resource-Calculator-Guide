# Panduan Komandan Pengumpul

**[Español](README_es.md) | [English](README_en.md) | [Português](README_pt.md) | [Tiếng Việt](README_vi.md) | Bahasa Indonesia | [Français](README_fr.md)**

Panduan cepat untuk menaikkan komandan pengumpul dengan efisien.

## Urutan rekomendasi

1. Sarka
2. Gaius Marius
3. Centurion
4. Joan of Arc
5. Matilda / Cleopatra / Ishida / Seondeok

## Progres yang disarankan

1. Naikkan Sarka dan Gaius ke level 20.
2. Naikkan Centurion ke level 20.
3. Saat Balai Kota 17, naikkan Joan ke level 20.
4. Naikkan 5 komandan pengumpul ke level 27.

## Target kapasitas

- Target menengah: 630000 per barisan.
- Target akhir: 1260000 per barisan.

## Rekomendasi utama

- Gunakan komandan pengumpul sebagai komandan utama.
- Prioritaskan talenta gathering sebelum talenta tempur.
- Latih pasukan untuk menambah kapasitas angkut.
