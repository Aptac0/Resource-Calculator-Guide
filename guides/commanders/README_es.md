# Guía de Comandantes Recolectores

**Español | [English](README_en.md) | [Português](README_pt.md) | [Tiếng Việt](README_vi.md) | [Bahasa Indonesia](README_id.md) | [Français](README_fr.md)**

Guía resumida para subir comandantes de recolección de forma eficiente.

## Orden recomendado

1. Sarka
2. Cayo Mario
3. Centurión
4. Juana de Arco
5. Matilde / Cleopatra / Ishida / Seondeok

## Progresión sugerida

1. Llevar Sarka y Cayo a nivel 20.
2. Llevar Centurión a nivel 20.
3. Al llegar al Ayuntamiento 17, subir Juana a nivel 20.
4. Subir los 5 recolectores a nivel 27.

## Objetivos de capacidad

- Objetivo intermedio: 630000 por marcha.
- Objetivo final: 1260000 por marcha.

## Recomendaciones clave

- Usa los recolectores como comandantes principales.
- Prioriza talentos de recolección antes que combate.
- Entrena tropas para aumentar la capacidad de carga.
